js第四天知识点总结

自执行匿名函数

  • 格式记住就行,之后会用的很多
  • 什么时候用？ 没必要声明变量的时候、
  • 或者避免变量全局污染的时候 、 划分工作空间

不死神兔-斐波那契数列

  • 递归调用问题
  • 递归注意问题:如果数量太大,容易出现内存溢出问题

求闰年平年天数问题

函数参数类型

  • 基本数据类型 number string boolean
  • 对象数据类型 array object
  • 注意:作为参数传递的区别!
  • 基本数据类型在函数内部会被创建一个副本,不会影响外部的数据
  • 对象数据类型传递的是该对象的一个引用(内存地址),所以修改的话是修改同一个对象.
  • json形式作为对象传值 会影响原来的值

对象定义的方法

  • 1.自变量形式定义对象 简单
  • 2.通过Object定义对象 Object是所有对象的顶级对象
  • 3.通过自定义函数形式定义对象
  • js中 万物皆对象 对象又皆函数
  • 注意：如果采用函数形式定义对象 首字母必须大写!

自定义函数的补充

  • this需要记住两条:
  • this指向调用者
  • 如果有new关键字,那么this指向new出来的对象本身.

json

  • json的值可以是什么类型？
  • 基本数据类型，对象类型，数组类型、json类型
  • 基于json的值可以为任意（function除外）,所以json可以描述非常复杂的数据格式
  • JSON.stringify()==> 把对象转换为json类型
  • json 和json对象 和对象之间的关系
  • JSON.parse(userJsonStr) ==> 把json的字符串类型 转换为对象类型

json遍历

  • 数组的遍历有两种形式 一种是for(var i = 0 ; i < arr.length ; i ++
  • for(var a in arr){….}
  • 注意：取对象的属性一般使用.的形式，尤其注意如果属性是一个变量或者字符串，那么
  • 需要使用[xx]的形式来取

数组高级内置方法

    1. toString 把数组转换为字符串
    1. sort 排序 sort参照的是字典排序
  • 作业：var eNameArr = [‘zhangsan’,’machale’,’john’,’scott’,’john’,’john’];
  • 写个方法 要求 接受数组和字符串 取出该数组中所有该字符串的下标

数组高级内置方法补充

  • indexOf 获取某个值的下标位置
  • slice(start,end) [start,end) 获取 不影响原来元素
  • splice(start,count) 从下标为start开始，截取count个元素,影响原来元素

清空数组

字符串操作

  • charAt 取某个下标的值 空格也算字符
  • substr 用的很多! 一般用于字符串截取
