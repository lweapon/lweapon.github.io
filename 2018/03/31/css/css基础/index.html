一 标签表现形式分类

块状元素

独占一格
宽高起作用

行内元素

可以在同一行显示
宽高不起作用
如果同一行放不下 多出部分将会换行

行内块元素

可以在同一行显示
宽高起作用
不会自动换行，但是宽高起作用 例如： img input select

块状元素和行内标签是可以相互转化的

块状标签转行内标签

内嵌式写法

代码可维护性较差，没有实现css代码与html结构的完全分离
影响的范围只有当前页面

内部样式

外联样式的写法

代码的可维护性高，css与html结构完全分离，形象范围广，css与html结构完全分离。

三 css特性

层叠性

样式的覆盖。样式的层叠性与样式的调用顺序没有关系，与样式的定义顺序有关，发生前提是：样式冲突。

继承性

继承性发生的前提是标签之间输入一整嵌套关系
文字的颜色可以实现继承
文字的大小可以实现继承
字体可以实现继承
行高可以实现继承
与文字有关的都可以实现继承

特殊性

优先级

默认样式<标签选择器<类选择器<id选择器<行内样式<!imporant

第一等：内联样式，如：style=“”，权值为1000
第二等：id选择器，如：#content，权值为100
第三等：类选择器，如content ，权值为10
第四等:标签元素选择器，如div p 权值为1
默认样式：0；
important：1000

优先级的特点

权重会叠加 继承权重为0

1. 文档流

  • 块元素独占一行
  • 行内元素在一行上显示
  • 自上而下

    2. 浮动(float)

  • 用法:float:left|right
  • 特点：
  • 浮动起来
  • 不占位置（脱标）

    如果多个盒子都使用了浮动，会在一行上显示。
    给行内元素使用浮动，可将行内元素转换成行内块。
    尽量用display转换行内块。

  • 作用：

    文本绕图

制作导航栏

网页布局

清除浮动

  • 清除不是删除浮动
    清除浮动是清除浮动产生的不良影响。
    注意:不给外边的父盒子设置高度时，里边的子盒子浮动时，会让页面页面布局产生混乱。

方法：clear:left|right|both

  • 使用最多的是clear:both;

额外标签法

在浮动元素后边加一个标签，清除浮动。

  • Overflow:hidden;
  • 在外边的父盒子上使用：overflow:hidden;
  • 如果有内容在盒子外，会被裁掉。

伪元素清除浮动

  • Overflow

  • Overflow:hidden; 会将出了盒子的内容裁掉

  • Overflow：auto;

  • 当内容出了盒子之外，会自动生成滚动条，如果没有内容之外，则不生成滚动条。
  • Overflow:scroll;
  • 不管内容有没有出盒子，都会生成滚动条。
  • overflow: visible;
  • 内容出了盒子会显示，不生成滚动条。

3. 定位

Left right top bottom

  • 3.1 静态定位(Position:static;)

    静态定位就是标准流。

  • 3.2 绝对定位（position:absolute;）

    特点

    给元素设置绝对定位，从浏览器左上角出发设置位置。
给元素设置了绝对定位，元素会不占位置（脱标）
当盒子发生嵌套关系时，如果父集元素没有设置定位，子盒子设置了绝对定位并赋值，子盒子的位置从浏览器左上角出发。
当盒子发生嵌套关系时，如果父集元素设置了绝对定位，子盒子设置了绝对定位并赋值，子盒子的位置从父集元素左上角出发。

    绝对定位会将行内元素，转换为行内块。

  • 3.3 相对定位(position:relative;)

特点：
给元素设置了相对定位，位置是从自身位置出发。
给元素设置了相对定位，还占据原来的位置（不脱标）。
子绝父相（工作中使用最多），子元素绝对定位，父元素相对定位。
给元素设置相对定位，不能将行内元素转换为行内块元素。

  • 3.4 固定定位（position:fixed;）
  • 特点:

    固定定位不占据原来的位置，会脱标。
    给元素设置固定定位之后，元素位置从浏览器左上角出发。
    可以将行内元素转换为行内块元素。

  • 3.5 定位的盒子居中显示

5. 规避脱标流

尽量使用标准流。
标准流解决不了的使用浮动。(排版)
浮动解决不了的使用定位。 (小图标等)

6. css隐藏元素常用属性

什么时候用浮动什么时候用定位

    1. 能用标准流（没有脱标）解决就不用浮动
    1. 解决不了就考虑有浮动（页面布局类型，“不完全脱标”）
    1. 浮动解决不了用定位（小图标，完全脱标，不影响内容）
