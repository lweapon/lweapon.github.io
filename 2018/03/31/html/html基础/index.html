    图片标签

相对路径和绝对路径

相对于文件自身出发，就是相对路径。

  • 文件和图片（html文档）在同一个目录(文件夹) ，直接写文件名

  • 图片（html文档）在文件在下一级目录里。文件夹名称+/+图片（html）名称

  • 图片（html）在文件的上一级目录里，..+/+图片（html）名称
  • 图片在文件的上一级的其他目录里，../文件夹名称/图片名称
  • 文件和图片（html文档）在同一个目录(文件夹) ，直接写文件名。

    超链接

  • href 去往的路径（跳转的页面） 必写属性
  • title 提示文本 鼠标放到链接上显示的文字 很多标签都有title属性
  • target=”_self” 默认值 可以不写 在自身页面打开（关闭自身页面，打开链接页面）
  • target=”_blank” 打开新页面 （自身页面不关闭，打开一个新的链接页面）
  • target = “_parent” 父类窗口打开

  • target = “_top” 在顶级页面打开
